# Power BI Learning Dataset

This repository contains 3 related CSV files designed for learning Power BI data modeling, visualization, and analysis.

## Dataset Overview

### Files Included:
- **customers.csv** - Customer information and demographics
- **products.csv** - Product catalog with pricing and categories  
- **sales.csv** - Sales transactions linking customers and products

### Relationships:
- Sales → Customers (CustomerID)
- Sales → Products (ProductID)

## Learning Tasks

### Task 1: Data Import and Modeling
- [✓] Import all 3 CSV files into Power BI Desktop
- [✓] Create relationships between tables using CustomerID and ProductID
- [✓] Verify data types and format columns appropriately
- [✓] Create a date table and establish relationship with SaleDate

### Task 2: Basic Visualizations
- [✓] Create a sales trend chart by month
- [✓] Build a bar chart showing sales by product category
- [✓] Design a pie visualization showing sales by customer location
- [✓] Create a table showing top 10 customers by revenue

### Task 3: Advanced Analytics
- [✓] Calculate total revenue, profit margin, and average order value using DAX
- [ ] Create a measure for Year-over-Year growth

### Task 4: Interactive Dashboard
- [ ] Create slicers for Date, Region, and Customer Type
- [ ] Build a comprehensive sales dashboard with multiple visuals
- [ ] Add drill-through functionality from summary to detail views
- [ ] Implement conditional formatting based on performance metrics

### Task 5: Data Analysis Expressions (DAX)
- [ ] Create calculated columns for profit and profit margin
- [ ] Write measures for running totals and moving averages
- [ ] Implement time intelligence functions (YTD, QTD, MTD)
- [ ] Build ranking measures for top performers
